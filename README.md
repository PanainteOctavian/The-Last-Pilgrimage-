## How to Run the Game

### Prerequisites
- Java JDK 8 or higher
- IntelliJ IDEA

### Installation Steps

1. **Extract the ZIP archive**
   - Unzip the downloaded file to your preferred location

2. **Open in IntelliJ IDEA**
   - Go to **File â†’ Open**
   - Navigate to the extracted folder `the-last-pilgrimage-main`
   - Select the **pom.xml** file
   - When prompted, choose **"Open as Project"**

3. **Configure Project Structure**
   - Right-click the `resources` folder and mark it as **"Resources Root"**
   - Right-click the `src` folder and mark it as **"Sources Root"**

4. **Run the Game**
   - Locate the `Main.java` file in `src/main` directory
   - Right-click and select **"Run Main.main()"**

---

### Explanation
This is a short Java game project that uses Maven as its build system. The `pom.xml` file contains all project dependencies and configuration. Marking the folders correctly ensures that resources (like images and sounds) and source code are properly recognized by IntelliJ.
